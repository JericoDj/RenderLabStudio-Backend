# 🤖 AI — Postman Guide

> **Base URL:** `http://localhost:3000/api/ai`

---

## 1. Improve Prompt

Enhances a user's raw prompt into a more detailed, optimized version.

| Field | Value |
|---|---|
| **Method** | `POST` |
| **URL** | `{{baseUrl}}/api/ai/improve-prompt` |
| **Headers** | `Content-Type: application/json` |

**Body (raw JSON):**
```json
{
  "prompt": "a cat sitting on a chair"
}
```

**✅ Success (200):**
```json
{
  "improvedPrompt": "A fluffy orange tabby cat sitting gracefully on a vintage wooden chair, soft natural lighting, photorealistic, 4K resolution"
}
```

**❌ Error (400):** `{ "message": "Prompt is required" }`

---

## 2. Chat

Sends a message within a session. The AI responds, and preferences may be auto-updated based on the prompt.

| Field | Value |
|---|---|
| **Method** | `POST` |
| **URL** | `{{baseUrl}}/api/ai/chat` |
| **Headers** | `Content-Type: application/json` |

**Body (raw JSON):**
```json
{
  "sessionId": "SESSION_OBJECT_ID",
  "prompt": "I want all images in anime style from now on",
  "modelProp": "gemini"
}
```

> - Replace `SESSION_OBJECT_ID` with a real session `_id`.  
> - `modelProp` accepts `"gemini"` or `"openai"` (defaults to OpenAI if omitted).

**✅ Success (200):**
```json
{
  "response": "Sure! I've noted your preference for anime style...",
  "preferencesUpdated": true,
  "preferences": {
    "video": {},
    "image": { "style": "anime" },
    "chat": {}
  }
}
```

**❌ Error (400):** `{ "message": "Session ID and prompt are required" }`  
**❌ Error (404):** `{ "message": "Session not found" }`

---

## 3. Signal Check

Checks if a prompt would trigger a preference change — without actually sending a chat message.

| Field | Value |
|---|---|
| **Method** | `POST` |
| **URL** | `{{baseUrl}}/api/ai/signal` |
| **Headers** | `Content-Type: application/json` |

**Body (raw JSON):**
```json
{
  "sessionId": "SESSION_OBJECT_ID",
  "prompt": "Switch to 4K resolution for videos"
}
```

**✅ Success (200):**
```json
{
  "signal": true,
  "suggestedChanges": {
    "video": { "resolution": "4K" }
  }
}
```

> If no preference change is detected: `{ "signal": false, "suggestedChanges": null }`
